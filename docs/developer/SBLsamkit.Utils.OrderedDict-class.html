Class OrderedDict

source code

object --+    
         |    
      dict --+
             |
            OrderedDict

Custom dict subclass which stores each key that's added in an ordered list. However, it doesn't support initializing with a dict, which itself by definition will be unordered. Provides basic ordered dict abilities on python older than 2.7
