Class SAMPair

source code

object --+
         |
        SAMPair

A class describing a read pair. Both reads of a pair are stored, and some pair-specific attributes are set, such as the computed distance between their start positions.

Returns a copy of the self with the appropriate strands reverse-complemented for output. Note that we assign the altered reads to the _read attribute, which is then always assumed to contain the sequence in "ready for output" orientation.

writeFASTQ(self, outputfiles, gzipFlag=False, closeWhenDone=True)

source code 

Basic output method - overridden in the Operators subclasses to add proper-pair handling. NOTE: This basic method does NO reverse-complement processing. If outputfiles[1] is None, both records will be written to the same file (outputfiles[0]), i.e. a shuffled output file.

__open__(self, filename, flags, gzipped=False)

source code 

Opens a file handle with the specified flags. If gzipped is True, the filename will have '.gz' appended to it and gzip.open() will be used instead of open(). In either case, returns the open file handle For maximum portabililty, the filename may also be an open filehandle (or filehandle-like object such as a StringIO object) in which case it is returned unchanged (and in that case the flags and optional gzipped argument are ignored)
