addRunSection(self, pct, pctLim, inputfile, outputdir, outputfiles, rangefiles, statsDict)

source code 

Sets up the run subsection and inserts it into the beginning of the ELN entry. This allows us to set up the run subsection at any time before writing the file and it will still be at the beginning of the ELN entry.
