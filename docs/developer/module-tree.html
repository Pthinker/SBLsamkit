Module Hierarchy

  • SBLsamkit: samkit 4.x modules.
    • SBLsamkit.Exceptions: Custom exceptions, each one accepts an optional second argument in the initializer, a stream name.
    • SBLsamkit.Info: Standard Info file classes.
    • SBLsamkit.MultiProcess: Classes and functions specifically for multi-processing
    • SBLsamkit.Operators: Constants defining operators and location and strand selector strings
    • SBLsamkit.Parser: Command line and INI file parser classes and functions
    • SBLsamkit.Processor: Main driver functions for samkit4
    • SBLsamkit.SAM: Classes and functions related to SAM Pairs
    • SBLsamkit.SBLhtseq: Custom SAM_Alignment classes and functions which enhance the functionality of the HTSeq python package
    • SBLsamkit.Stats: Statistics functions designed to return lists of strings which can be easily added to an InfoFile's ELN object
    • SBLsamkit.Stream: Base stream class
    • SBLsamkit.Utils: Miscellaneous utility functions and classes
